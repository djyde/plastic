# Plastic

A block-based bi-directional note app

![](https://assets.djyde.com/uPic/nA94Ev.png?x-oss-process=style/high-optimize)

# Build


## Development

```bash
yarn

yarn web # first, run the frontend build

yarn app # second, run electron app
```

## Distribute

```bash
cd app && yarn
cd ..
yarn dist
```
